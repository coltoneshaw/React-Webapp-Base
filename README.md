# React Webapp Base

This package includes React, Typescript, Webpack, and Jest.

# Commands

## Development
Starts a dev server on port 9000

```bash
yarn dev
```

## Test (Jest)

This runs jest across the project. This sets `NODE_ENV` to `development`.

```bash
yarn jest
```

## Build Production
Builds your code and outputs it to `./release`

```bash
yarn production
```
